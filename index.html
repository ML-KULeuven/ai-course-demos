N-Puzzle is a web app for learning about graph-based search algorithms.

It tries to go from an initial 3x3 state to a desired 3x3 state by moving one tile into an empty spot. The search tree is visualized and you will be able to tell what happens one step at a time.

Try it out and find more information using the following link:
